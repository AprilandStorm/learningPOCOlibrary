# 构建一个基于 POCO C++ Libraries 的「远程设备监控与控制系统」
## 功能需求
|模块|功能说明|
|-----|-----|
|设备端 Device|模拟若干 IoT 设备（温度传感器 / 开关 / 小车等），定时上报数据|
|服务器 Server|使用 POCO HTTP + WebSocket 模块，接收设备数据并转发给控制端|
|控制端 Client|可以查看实时数据、发送控制指令|
|日志与配置系统|使用 Poco::Logger 和 Poco::Util::IniFileConfiguration|
|线程模型|多线程并发管理设备连接，使用生产者–消费者队列同步消息|

## 最终运行效果
- 控制端可实时显示多设备状态；
- 可以远程控制设备（开/关、设置参数）；
- 有日志文件记录每次交互事件。
